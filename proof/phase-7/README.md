## Outcome

PASS — the public learning app turns synthetic TxLINE recordings and the finalized mainnet escrow receipt into six distinct, interactive learning surfaces. No restricted feed data, credential, signer, or authorization material is bundled.

## Learning surfaces

- Start here: evidence journey, public-data boundary, and fixture catalog.
- Replay lab: fixture switching, record stepping, timeline scrubbing, channel density, and envelope inspection.
- Strategy studio: side and score slicing with immediate predicate truth and compiler output.
- Proof anatomy: interactive Merkle path, receipt fields, and trust ladder.
- Settlement: finalized mainnet transaction river, compute receipt, explorer drill-down, and safety invariants.
- SDK map: spatial module explorer with import guidance and typed-boundary guarantees.

## Dependency release gate

The published SDK no longer ships `@solana/spl-token` as a runtime dependency. A minimal internal Token-2022 ATA helper is byte-for-byte tested against SPL Token for program IDs, address derivation, account metas, and instruction data. SPL Token remains dev-only for equivalence tests and deployment scripts.

## Boundary

This is a hackathon learning experience, not an audited real-money wagering product. Its match fixtures are intentionally synthetic; its settlement view uses public finalized chain receipts only.
