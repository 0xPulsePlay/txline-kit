# Security

Please report vulnerabilities privately through GitHub's security-advisory flow for `0xPulsePlay/txline-kit`. Do not open a public issue for an undisclosed vulnerability.

TxLINE Kit is hackathon integration software and has not received a third-party security audit. The demo escrow uses valueless Token-2022 units and is not suitable for real-money custody. Consumers must independently review their transaction construction, signer authority, retry idempotency, proof assumptions, and settlement lifecycle.

Never include API credentials, wallet keys, authorization headers, restricted feed recordings, or personal data in a report attachment.
